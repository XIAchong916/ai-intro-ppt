# AI 知识框架科普
从模型 → 大模型 → Agent → 真实系统，一站式理解 AI 本质、能力与落地应用

## 项目介绍
这是一份面向 AI 初学者的体系化科普材料，以通俗语言、层层递进的逻辑，完整讲解 AI 核心概念与落地路径。

适合：AI 入门、内部培训、技术分享、科普宣讲。

## 内容结构
1. 基础认知：AI 本质、模型定义、发展历程、分类视角
2. 大模型与 Agent：智能形成、能力边界、Agent 闭环、真实智能研究、关键支撑
3. AI 真实系统落地：五层应用路径、人机协作、个人定位

## 适用场景
- 个人自学 AI 基础体系
- 企业/团队 AI 科普培训
